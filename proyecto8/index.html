Instrucciones para el proyecto:

Registro de empleados usando objetos y constructores con estos datos:

  • Código numérico de empleado
  • Nombre
  • Apellido
  • Año de nacimiento
  • Cargo

Debe tener estos 5 inputs con sus labels.

Luego un botón para cargar empleado, y otro para mostrar empleados.

Usar constructores para las fichas de empleados.

Con el segundo botón, mostrar en un alert a los empleados, con sus propiedades.

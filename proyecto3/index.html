Proyecto 3 del curso de JS y HTML

En esta página aplicaré lo que pide el profesor para el proyecto 3.
La idea es que en cada página de proyecto se aprecie el progreso en complejidad
de las características aprendidas durante el curso.

En este proyecto tan solo se piden estas características:

  • Sitio web de una página
  • 30 segundos
  • Responder 5 preguntas
  • Si falla: alert('Game Over') + Botón 'Reiniciar'
  • Si lo logra: alert ('fecha y respuestas') + Botón 'Listo'

Elementos indispensables para el ejercicio

  • Cuenta regresiva de 30 a 0
  • 5 preguntas
  • 5 inputs
  • Botón 'Listo' o 'Enviar'
  • Botón 'Reiniciar'

Funciones

  • Mostrar tiempo inicial
  • Actualizar el tiempo en pantalla
  • Avisar cuando acabe el tiempo
  • Mostrar alert con fecha y respuestas
  • Botón que reinicia el sitio

Disclaimer:

El sitio es extremadamente básico: no comprueba si las respuestas son correctas. Tan solo es una prueba de las funciones que mencionadas, no pretende ser una página con toda su aparente funcionalidad operativa.
